# ffmpeg utilites

You will find some scripts here for convience regarding ffmpeg, just use `--help` (`-h`) for usage.

## Dependencies
Requires ffmpeg
- Debian or it's variants i.e. Ubuntu
  
  ```bash
  apt install ffmpeg
  
  # use sudo if not root
  
  sudo apt install ffmpeg
  ```

- Arch
  ```bash
  pacman -S ffmpeg
  ```

- For other distros consult the docs
